# TIPE---Detection-automatique-des-plantes-dans-les-cultures

Ce projet a été réalisé dans le cadre de mon TIPE.
L’objectif est d’entraîner un modèle YOLOv8 pour différencier les feuilles de plantes (apple, blueberry, cherry, corn, soybean, tomato).


![Diaporama de présentation](docs/TIPE_BUC_Jade.pdf)

Le dépôt contient :
- mon script d’entraînement YOLO (yolo.py)
- un CNN simple (cnn.py) utilisé pour des tests préliminaires
- le dataset organisé au format YOLO
- les poids entraînés (cnn.pth)
