# TIPE---Detection-automatique-des-plantes-dans-les-cultures

Ce projet a été réalisé dans le cadre de mon TIPE.
L’objectif est d’entraîner un cnn pour différencier les feuilles de plantes (apple, blueberry, cherry, corn, soybean, tomato).


![Diaporama de présentation](docs/TIPE_BUC_Jade.pdf)

Le dépôt contient :
- un CNN simple (cnn.py) 
- le dataset
- les poids entraînés (cnn.pth)
- exemple transformation pour optimiser l'efficacite de l'entrainement (transformation.py)
- une fois le modele entrainer on peut tester le code avec une image choisie (fenetre.py)
